🏭 Magazijn Gereedschapsbeheer

Verbind met je Supabase-database om te starten. Volg de stappen hieronder als je nog geen database hebt.

Stap 1 — Database aanmaken in Supabase
  1. 1. Ga naar supabase.com en maak een gratis account
  2. 2. Klik op "New project", kies een naam en regio (bijv. West Europe)
  3. 3. Ga naar SQL Editor en plak de code hieronder, klik op "Run"
  4. 4. Ga naar Project Settings → API en kopieer de URL en anon key
-- Tabel: gereedschap
CREATE TABLE gereedschap (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  naam TEXT NOT NULL,
  categorie TEXT DEFAULT 'other',
  serienr TEXT,
  locatie TEXT,
  status TEXT DEFAULT 'beschikbaar',
  defect_omschrijving TEXT,
  defect_door TEXT,
  defect_datum DATE,
  aangemaakt_op TIMESTAMPTZ DEFAULT now()
);

-- Tabel: uitleningen
CREATE TABLE uitleningen (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  tool_id UUID REFERENCES gereedschap(id) ON DELETE CASCADE,
  persoon TEXT NOT NULL,
  afdeling TEXT,
  datum DATE NOT NULL DEFAULT CURRENT_DATE,
  verwacht_terug DATE,
  terug_op DATE,
  opmerking TEXT,
  aangemaakt_op TIMESTAMPTZ DEFAULT now()
);

-- Rij-beveiliging uitschakelen (iedereen mag lezen/schrijven)
ALTER TABLE gereedschap ENABLE ROW LEVEL SECURITY;
ALTER TABLE uitleningen ENABLE ROW LEVEL SECURITY;

CREATE POLICY "open_gereedschap" ON gereedschap FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "open_uitleningen" ON uitleningen FOR ALL USING (true) WITH CHECK (true);

Je sleutels worden lokaal opgeslagen in je browser en nooit naar externe servers gestuurd.
